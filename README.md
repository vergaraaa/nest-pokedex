# Run in development

1. Clone repo
2. Run

```
yarn installl
```

3. Have Nest CLI installed

```
npm i -g @nestjs/cli
```

4. Run db

```
docker-compose up -d
```

## Tech Stack

- MongoDB
- Nest
- Docker
